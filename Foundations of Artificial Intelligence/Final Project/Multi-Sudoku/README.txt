The main program is run using the command

python multi-sudoku.py

This program will run example puzzles of 2x2 Multi-Sudoku, 3x3 Multi-Sudoku, Sohei-Sudoku, and Samurai Sudoku. The example puzzles included in the repository are

2x2_sudoku_q.txt
3x3_sudoku_q.txt
sohei_sudoku_q.txt
samurai_suoku_q.txt

The solutions to these puzzles are 
2x2_sudoku_a.txt
3x3_sudoku_a.txt
sohei_sudoku_a.txt
samurai_suoku_a.txt

The main program will solve the puzzles and check against these solution files.
